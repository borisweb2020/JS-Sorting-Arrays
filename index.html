Тестовое задание FOCUSSTART-24944

Описание:

Входные массивы целых чисел должны быть записаны в файле JSON-формата, расположенного в папке file с именем arrays.json. Пример структуры файла показан в разделе Входящие массивы.

Перед сортировкой массивов по возрастанию, они будут проверены на содержание "инородных" типов данных. Например, если массив будет содержать элементы типа string, boolean, NaN, то такой элемент будет удален из массива. После проверки все массивы будут объеденены в один, который и будет отсортирован методом Сортировка слиянием, поэтому исходный порядок сортировки не важен.
